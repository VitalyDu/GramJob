Layout:

1. Шапка:
   1. Меню:
      1. Вакансии
      2. Компании
      3. Мои резюме (только у авторизованных пользователей)
      4. База резюме (только у авторизованных пользователей, с тарифом Max)
   2. Убрать badge с тарифом из шапки
   3. В мобильной версии кнопка «Язык интерфейса» должна вызывать не dropdown, а Drawer, в котором будет выбор языка
   4. В dropdown пользователя добавить перед firstname иконку пользователя
   5. В dropdown пользователя, после firstname добавить lastname (если имеется)
   6. Справа от firstname + lastname в dropdown пользователя добавить SubscriptionBadge
2. BottomNav:
   1. GramJob (только в telegram)
   2. Вакансии
   3. Отклики
   4. Избранное
   5. Кабинет
3. В телеграме:
   1. На всех страницах, сверху справа должны быть иконки:
      1. Avatar:
         1. Открывает Drawer (в нём представлено меню):
            1. Avatar + firstname + lastname (если имеется) + SubscriptionBadge (ссылка на страницу профиля)
            2. Уведомления (ссылка на страницу уведомлений)
            3. Подписка (ведёт на страницу «Подписка)
            4. Язык (вызывает Drawer, в нём выбор языка)
      2. Уведомления (ссылка на страницу уведомлений), (также необходим каунтер, если имеются непрочитанные уведомления)
      3. Globe (открывает Drawer, в нём выбор языка)
