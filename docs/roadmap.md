# GramJob — Feature Roadmap

Предложения по новым функциям и расширению платформы.
Порядок не означает приоритет — требует обсуждения.

---

## AI & Умные функции

### AI-matching вакансий и кандидатов

Автоматическое сопоставление вакансий с резюме на основе навыков, опыта и предпочтений.

- Работодатель видит рейтинг откликов
- Кандидат видит % совпадения с вакансией
- Рекомендации вакансий на главной

### AI Cover Letter

Генерация сопроводительного письма на основе резюме и описания вакансии.

- Кнопка «Сгенерировать» в форме отклика
- Редактирование перед отправкой
- Несколько вариантов на выбор

### Salary Insights

Рыночные данные о зарплатах по специализации и региону.

- Показывать диапазон «рынка» при создании вакансии
- Раздел «Зарплаты» с аналитикой для кандидатов

## Доверие и верификация

### Verified Company Badge

Верификация компаний через документы.

- Загрузка юридических документов
- Ручная проверка модераторами
- Бейдж ✓ Verified на карточке

### Company Reviews

Отзывы сотрудников о работодателях.

- Анонимные отзывы (только для верифицированных пользователей)
- Рейтинг: 1-5 звёзд + текст
- Работодатель может ответить на отзыв

## Инструменты рекрутера

### Recruiter CRM

Управление воронкой кандидатов для работодателей.

- Kanban-доска по статусам откликов
- Заметки к кандидатам
- Теги и пометки
- Групповые действия

### Interview Scheduling

Интеграция с календарём для записи на интервью.

- Работодатель предлагает слоты
- Кандидат выбирает удобное время
- Автоматическое уведомление в Telegram + Email

### Team Collaboration

Несколько сотрудников компании управляют вакансиями.

- Роли: Admin, Recruiter, Viewer
- Совместный доступ к откликам

## Профиль кандидата

### Portfolio / Work Samples

Раздел для примеров работ в резюме.

- Ссылки на проекты (GitHub, Behance, etc.)
- Загрузка файлов (PDF, изображения)
- Встроенные превью

### PDF-экспорт резюме

Скачать резюме в формате PDF.

- Несколько шаблонов оформления
- Логотип GramJob как watermark (бесплатно) или без (Pro+)

### Video Introduction

Короткое видео-знакомство в резюме (до 60 сек).

- Загрузка или запись прямо в приложении
- Особый бейдж «Video» на карточке резюме

## Монетизация и рост

### Referral Program

Реферальная программа.

- Уникальная ссылка для каждого пользователя
- Бонус: +N дней подписки за привлечённого пользователя
- Пригласи 3 работодателей → получи Pro на 1 месяц

### Job Alerts (Saved Searches + Email Digest)

Возврат модуля сохранённых поисков (удалён из v1) + email-дайджест с новыми вакансиями под сохранённые запросы.

- Каналы: Telegram + Email
- Настройка частоты: ежедневно / еженедельно

## Технические улучшения

### Advanced Search

- Fuzzy search по навыкам
- Синонимы (JavaScript = JS = Node.js)
- Автодополнение (autocomplete) для строки поиска и навыков
- Elasticsearch / Meilisearch

### Audit Log

Полный лог действий пользователей для compliance.

### Export данных

- Работодатель: выгрузка откликов в Excel
- Кандидат: выгрузка своих данных (GDPR)

### Webhooks для работодателей

Уведомления о новых откликах через HTTP webhook (для интеграции с ATS).
